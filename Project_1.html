Laptop Price Prediction

Problem Statement

The laptop market is influenced by numerous factors including brand reputation, hardware specifications, market demand, and technological advancements. Predicting laptop prices accurately is challenging due to the complexity and variability of these factors. This project leverages a data-driven approach using machine learning algorithms to develop a robust model for accurate laptop price predictions, providing valuable insights for consumers and sellers alike.


Dataset Description:

  • Company: Name of the company
  • TypeName: Type of laptop (e.g., Notebook, Gaming, Ultrabook, 2 in 1 Convertible, Workstation, Netbook)
  • Inches: Size of the laptop (floating value)
  • ScreenResolution: Laptop screen resolution
  • Cpu: Processor (CPU) specifications
  • Ram: Amount of RAM
  • Memory: Storage capacity/type (e.g., SSD, HDD)
  • Gpu: Graphics processor (GPU) specifications
  • OpSys: Operating system installed
  • Weight: Laptop weight (kg or lbs)
  • Price: Target variable for prediction

Libraries and Algorithms Used:

  • Data Manipulation and Analysis: pandas, numpy
  • Mathematical Functions: math
  • Data Visualization: matplotlib, seaborn
  • Machine Learning Algorithms:
    • Linear Models: Linear Regression, Ridge, Lasso
    • Nearest Neighbors: KNeighborsRegressor
    • Tree-Based Models: Decision Tree Regressor, Random Forest Regressor, Gradient Boosting Regressor, AdaBoost Regressor, Extra Trees Regressor
    • Support Vector Machines: SVR
    • Boosting Algorithms: XGBRegressor
  • Model Evaluation Metrics: mean_absolute_error, mean_squared_error, r2_score
  • Train-Test Split: train_test_split from sklearn

Data Cleaning and Insights:

  • Removed units (e.g., GB from RAM column, kg from Weight column) and changed data types accordingly.
  • Most common laptop type is Notebook (55.8%), likely due to its affordability.
  • Some gaming laptops are priced around 3 lakhs.
  • Created a new column, PPI (Pixels Per Inch), to measure display sharpness and clarity.
  • Touchscreen laptops tend to be priced higher, with 2 in 1 Convertible laptops often being touchscreen.
  • Ultrabooks and other types also include some touchscreen models.
  • Correlation between resolution, PPI, and price is notable. Removed redundant columns after PPI creation.
  • Most people prefer laptops with 8GB RAM, likely due to budget considerations.
  • Nvidia processors are generally more expensive.

Preprocessing Techniques Used:

  • One-Hot Encoding: OneHotEncoder from sklearn.preprocessing
  • Column Transformation: ColumnTransformer from sklearn.compose
  • Label Encoding: LabelEncoder from sklearn.preprocessing
  • Feature Scaling: StandardScaler from sklearn.preprocessing

Final Conclusion:

After evaluating several models, XGBoost demonstrated the highest accuracy of around 90%. Therefore, XGBoost was selected as the final model due to its superior performance compared to other machine learning algorithms used in this project.
